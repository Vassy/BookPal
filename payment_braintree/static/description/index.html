Detailed Features List

Below is the list of features BrainTree Payment Acquirer.

Payment Method For Odoo Your Website

  • The module facilitates quick payments for customers on their online orders on Odoo website using Braintree Payment Services Gateway.
  • Braintree Payment shows up as one of the payment methods on the check out page on Odoo website.

Supports multiple cards for payment process

  • Braintree Payment supports MasterCard, Visa, American Express, JCB, Discover and Many more.
  • Customer can delete the save card from website frontend From Manage payment methods From their My Account section.

Save card details for future purposes

  • User can select save my payment details option on frontend to save card details for future use.
  • User can simply select the save card and click on paynow button to place order Reference.

Check token for saved cards on braintree platforms

  • You can check the token for saved card on braintree platform.

Merchant Authentication

  • One step Braintree Payment integration with Odoo.
  • The admin just needs to enter Merchant ID, Public Key, and Private Key from Braintree in Odoo.

Secure Payments

  • After selecting Braintree payment method on the website, the customer enters his/her card details in the pop-up and follow the steps.
  • The transaction is completed on the Odoo website itself.
  • The customer is notified after the transaction is complete.

Track Transaction History

  • The Odoo admin can see the transaction details of the orders in the Odoo backend.
  • The order form shows the details of the mode of payment used by the customer to make the payment for his/her order.
  • The list of the transaction is also maintained in the Braintree merchant portal.

Enable 3D Secure Feature

  • To further increase security, the user can enable 3D secure feature for Braintree Payments on the Odoo website.

Multi-Currency Support

  • Add multiple currency transaction to the Braintree in Odoo.
  • Once done, the transaction is automatically processed according to the currency of website (if that currency is added to merchant account.
  • The list of the transaction is also maintained in the Braintree merchant portal.

Manage the payment configuration from the backend

  • Admin can enable/disable this feature for braintree payment acquirer configuration in backend.
  • Payment Token will create in backend for every saved card and admin can check the payments made by the save card.
